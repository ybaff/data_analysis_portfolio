# Project Overview

This project analyses an e-commerce dataset to understand business performance, identify key revenue drivers, and uncover actionable insights.

The analysis focuses on answering business questions such as:

What is the total revenue generated?

Which products generate the most revenue?

How does revenue change over time?

Which days or months generate the most sales?

What insights can help improve business performance?

The project demonstrates a typical data analyst workflow, combining Python for data preparation and SQL for business analysis.

# Tools Used

Python (Pandas) – data loading and feature engineering

SQLite / SQL – querying and analysing the dataset

Jupyter Notebook – project environment

# Dataset

The dataset used in this project contains transactional e-commerce data including:

Invoice numbers

Product descriptions

Quantity sold

Unit price

Invoice date

Customer location

Each row represents a product purchase within an order.

# Data Preparation

The dataset was first loaded into Python using pandas.

A Revenue column was created to calculate the total revenue generated by each transaction:

df["Revenue"] = df["Quantity"] * df["UnitPrice"]

The processed dataset was then stored in a SQL database for analysis.

# SQL Analysis

Several SQL queries were used to explore business performance.

Total Revenue
SELECT SUM(Revenue)
FROM ecommerce

This calculates the total revenue generated by the business.

Top Revenue Generating Products
SELECT
Description,
SUM(Revenue) AS total_revenue
FROM ecommerce
GROUP BY Description
ORDER BY total_revenue DESC

This identifies the products generating the highest revenue.

Revenue by Month
SELECT
strftime('%Y-%m', InvoiceDate) AS month,
SUM(Revenue) AS monthly_revenue
FROM ecommerce
GROUP BY month
ORDER BY month

This shows how revenue changes over time.

Revenue by Day of Week
SELECT
strftime('%w', InvoiceDate) AS weekday,
SUM(Revenue) AS revenue
FROM ecommerce
GROUP BY weekday

This helps identify the best performing days of the week.

# Key Insights

From the analysis we can observe several patterns:

A small number of products generate a large portion of total revenue.

Revenue fluctuates across different months, indicating possible seasonality.

Certain days of the week consistently produce higher sales.

These insights could help the business optimise:

product marketing

inventory planning

promotional campaigns
